### REQUIRED REPOSITORIES ###

```bash
$ sudo pacman -S hyprland hyprpaper waybar rofi-wayland zsh fastfetch cava 

```

### Required for characters to appear correctly ###
```bash
$ sudo pacman -S ttf-font-awesome otf-font-awesome ttf-jetbrains-mono-nerd noto-fonts-emoji
```

### ZSH ###
The name of the file “zshrc” was changed so that it could be viewed in the folder.
Remember to change its name back to “.zshrc” when you move it to its corresponding path.


### HYPRPAPER ###
Remember to modify the path in hyrpaper.conf so that it detects the image.
